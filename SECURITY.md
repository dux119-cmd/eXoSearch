# Security Policy

## Supported Versions

The following versions are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| main   | :white_check_mark: |

## Reporting a Vulnerability

Open a new issue to report a vulnerability.

The project authors will respond to the issue indicating if the vulnerability is accepted or
declined. If accepted, the authors will address the vulnerability at their next oppourtunity.
